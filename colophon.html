How I made this website?

This website is a github page.

The theme is forked from an awesome guy called Zach Holman. He made this with awesome receipies like HTML5, SCSS, Coffee Scripts, webfonts etc. Don't blame me if this website doesn't appear well on Internet Explorer

I really appreciate the excellent fonts produced by Symbolset which shows the social network symbols found on this websites. Holman's original page repo contains the copy of the social fonts he purchased but I really loved it and I purchased my own from their website.

This website is tracked by Google Analytics and currently not ads are displayed in this website.

Where I was all these years?

I have been blogging for years. Lot of technical tips I posted in my previous wordpress blog. Unfortunately I became so lazy to make better posts and was enjoying the traffic which was generated from my early popular posts. I decided to cannibalize the blog and start over something. The place I hosted my website was cheap but not so reliable. I wanted something simple, really hated FTPs, MySQLConsoles, backups and spammy plugings which has ruined my website once. Github is simple and plain. My posts are just few text files, I can take to anywhere. Also markup language of Github is awesome and not complications of HTML tags
